2. CASCADE CLEANLINESS (7 levels)
═══════════════════════════════════════════════════════════════

Level thresholds based on overall_cleanliness score:
  L7 Liberation:    cleanliness >= 0.90
  L6 Dharmic:       cleanliness >= 0.70
  L5 Awakening:     cleanliness >= 0.50
  L4 Seeking:       cleanliness >= 0.35
  L3 Confusion:     cleanliness >= 0.20
  L2 Suffering:     cleanliness >= 0.10
  L1 Crisis:        cleanliness < 0.10

Cleanliness Formula:
  base_cleanliness = avg(positive_ops) - avg(negative_ops)
  positive_ops: G_grace, S_surrender, P_presence, Co_coherence, W_witness, Aw_awareness
  negative_ops: At_attachment, R_resistance, F_fear, E_ego, M_maya, K_karma
  overall_cleanliness = base_cleanliness * flow_multiplier * grace_boost

Flow multiplier: 1.0 + (0.3 * flow_score) where flow = P * Co * (1-R)
Grace boost: 1.0 + (0.2 * G_grace) if G_grace > 0.7

6. UNITY METRICS
═══════════════════════════════════════════════════════════════

Unity Score (higher = more unified):
  unity_positive = avg(G_grace, S_surrender, P_presence, W_witness, Aw_awareness, L_liberation)
  unity_score = unity_positive * coherence_factor
  coherence_factor = Co_coherence * (1 - internal_conflict)
  internal_conflict = abs(positive_ops_variance)

Separation Score (higher = more separated):
  separation_negative = avg(At_attachment, E_ego, F_fear, R_resistance, M_maya)
  separation_score = separation_negative * fragmentation_factor
  fragmentation_factor = 1 + (At * E * 0.5)

Net Unity Index:
  net_unity = unity_score - separation_score
  Range: -1.0 (total separation) to +1.0 (total unity)

  Interpretation:
    > 0.7  : Non-dual awareness stable
    > 0.4  : Unity glimpses frequent
    > 0.0  : Transitional/seeking
    < 0.0  : Separation dominant
    < -0.5 : Deep fragmentation

  High-leverage operator = low current value + high potential impact

  Surrender Leverage:
    S_surrender < 0.4 AND At_attachment > 0.6

  Presence Leverage:
